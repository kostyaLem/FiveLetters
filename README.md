# FiveLetters
Копия игры Тинькофф 5 букв

### Стек технологий:
- #### Паттерны: 
  > ООП, DI, IoC, MVVM
- #### Технологии (.NET 6): 
  > WPF, HandyControl
- #### Дополнительно:
  > Нестандартный дизайн, режим игры для английских слов
